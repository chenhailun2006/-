#爱记账APP项目说明
```
爱记账APP使用到的技术有：angular2，ionic2，cordova，数据保存使用ionic的storage技术
```
#使用本项目

1.安装依赖包
```
$ npm install
```
2.启动项目
```
$ ionic serve
```
3.ionic常用命令
```
(1)新建组件
ionic g component <组件名称>

（2）新建页面
ionic g page <页面名称>

（3）新建服务
ionic g provider <服务名称>

（4）新建管道
ionic g pipe <管道名称>

（5）新建指令
ionic g directive <指令名称>
